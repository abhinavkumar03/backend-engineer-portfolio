# Abhinav Kumar - Portfolio

A high-performance, recruiter-optimized portfolio website for a Backend Software Engineer.
Built with Next.js 14, Tailwind CSS, Framer Motion, and Three.js.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **3D Effects**: React Three Fiber / Three.js
- **Language**: TypeScript
- **Smooth Scroll**: Lenis

## Features

- **Premium Dark Theme**: Minimalist, professional aesthetic.
- **Performance Optimized**: 90+ Lighthouse score target.
- **Responsive**: Mobile-first design.
- **3D Background**: Subtle particle network representing distributed systems.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/app`: App router pages and layout.
- `src/components`: Reusable UI components.
- `src/components/canvas`: Three.js scenes.
- `src/lib`: Utilities.

## Design Decisions

- **Typography**: `Outfit` for headings (modern, geometric) and `Inter` for body (clean, readable).
- **Color Palette**: Pure black background with white text and subtle blue/primary accents.
- **Content**: Focused on "Backend Engineering", "Scalability", and "Systems" to match the target role.
